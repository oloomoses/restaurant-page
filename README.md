## About The Project
This is a Javascript app that showcases my skills in DOM manipulation and use of webpack, by dynamically rendering a simple restaurant App.
I have used JavaScript to generate the entire contents of the website!

### Built With
Components of this project is listed below;

* Javascript - ES6
* VsCode
* Webpack
* EsLint
* css
